## Quick Import Instructions

1. Open Framer CMS → Blog Posts collection
2. For each post above:
   - Click "Add Item"
   - Copy/paste the fields from this template
   - For "Content" field, copy the full blog post from `blog-content-drafts.md`
   - Save and publish

3. After importing all 12 posts:
   - Go to `/blog` page and verify all posts appear
   - Click on each post to ensure proper rendering
   - Check that meta descriptions are showing in page source

## SEO Schema for Blog Posts

After importing, add this JSON-LD schema to each blog post page (in Page Settings → Custom Code → End of `<head>`):

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "BlogPosting",
  "headline": "[POST TITLE]",
  "description": "[META DESCRIPTION]",
  "author": {
    "@type": "Organization",
    "name": "Avakata Agency"
  },
  "publisher": {
    "@type": "Organization",
    "name": "Avakata Agency",
    "logo": {
      "@type": "ImageObject",
      "url": "https://avakata.agency/logo.svg"
    }
  },
  "datePublished": "[PUBLISH DATE in ISO format]",
  "dateModified": "[PUBLISH DATE in ISO format]",
  "mainEntityOfPage": {
    "@type": "WebPage",
    "@id": "https://avakata.agency/blog/[SLUG]"
  },
  "image": "[FEATURED IMAGE URL if applicable]",
  "articleSection": "[CATEGORY]",
  "keywords": "AI marketing, autonomous AI, [relevant keywords]"
}
</script>
```

Replace the bracketed placeholders with actual values for each post.
