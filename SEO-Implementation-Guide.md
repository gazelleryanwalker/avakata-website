## How to Implement in Framer

### 1. Page SEO Settings (Per Page)

In Framer, go to each page and click the gear icon (⚙️) > SEO Settings:

1. **Title Tag:** Enter the title from above
2. **Meta Description:** Enter the description from above
3. **Social Image:** Upload a relevant OG image (1200x630px)
4. **Custom Code (Head):** Add Schema.org JSON-LD from above

### 2. Add llms.txt File

#### Option A: Via Framer Code Files

1. In Framer, go to the Assets panel
2. Click "+ New" > "Code File"
3. Name it `llms.txt`
4. Paste the contents of the `llms.txt` file I created
5. This will be accessible at <https://avakata.agency/llms.txt>

#### Option B: Via Framer Site Settings

1. Go to Site Settings > Custom Code
2. Add a redirect rule or use a code override to serve the file

### 3. Add Sitemap

**In Framer:**

1. Go to Site Settings > SEO
2. Framer auto-generates a sitemap at `https://avakata.agency/sitemap.xml`
3. Verify it includes all your pages
4. If you need a custom sitemap, use the one I created above

### 4. Submit to Google Search Console

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add property: `https://avakata.agency`
3. Verify ownership (Framer provides verification methods)
4. Go to Sitemaps section
5. Submit: `https://avakata.agency/sitemap.xml`
6. Monitor indexing status

### 5. robots.txt

Framer auto-generates this, but verify it includes:

```text
User-agent: *
Allow: /
Sitemap: https://avakata.agency/sitemap.xml
```

## Performance Optimization

### Images

- Compress all images (use WebP format)
- Add `loading="lazy"` to below-the-fold images
- Use responsive images with srcset

### Fonts

- Preload critical fonts
- Use font-display: swap

### Scripts

- Defer non-critical JavaScript
- Minimize third-party scripts

### Core Web Vitals Targets

- **LCP (Largest Contentful Paint):** < 2.5s
- **FID (First Input Delay):** < 100ms
- **CLS (Cumulative Layout Shift):** < 0.1
